# Recipe App

A simple React recipe search app built with Vite and the Edamam Recipe API.

## About

This app allows users to search for recipes by entering a keyword. The app fetches recipe data from the Edamam API and displays recipe cards with images, labels, calories, and ingredients.

## Features

- Search recipes by keyword
- Fetch data from an external API
- Display recipe images
- Show recipe title and ingredients
- Responsive layout
- Built with React components and hooks

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- Edamam Recipe API


## API

This project uses the Edamam Recipe API.

To run this project, you need your own API ID and API Key from Edamam.
